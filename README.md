# 🐶 발바닥 구조대_Pawpatrol

### AI기반 실종글과 제보글의 이미지를 매칭, 잃어버린 반려동물을 구조하고 유기동물은 보호소가 아닌 임시보호 및 실제 주인을 찾아줄 수 있도록 하는 유기동물 구조 애플리케이션

## 나의 역할
- AI 유사도 분석 시스템 연동 및 Kafka 기반 비동기 아키텍처 설계, 실시간 제보 매칭 성능 최적화
- CICD 구축 (AWS, NaverCloud, Terraform, Docker)
